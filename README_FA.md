# Shade

یک کلاینت پراکسی پرمیوم و سبک برای macOS جهت هدایت ترافیک از طریق **Google Apps Script**. با رابط کاربری بومی (Native) و زیبا، به راحتی از سد سانسور عبور کنید.

## امکانات کلیدی

- **نمایشگر سرعت زنده**: مشاهده سرعت دانلود/آپلود و میزان کل دیتای مصرفی به صورت لحظه‌ای.
- **مینی‌اپ منوبار**: کنترل اتصال، تغییر وضعیت پراکسی سیستم و مشاهده سرعت، مستقیماً از نوار منوی macOS.
- **موتور پراکسی یکپارچه**: سرورهای داخلی SOCKS5 و HTTP.
- **نصب خودکار گواهی (HTTPS)**: تولید و نصب خودکار گواهی (Certificate) برای رمزگشایی بی‌دردسر ترافیک HTTPS.
- **مدیریت پروفایل**: تعریف چندین سرور مختلف و جابه‌جایی سریع بین آن‌ها.
- **اسکنر آی‌پی گوگل**: ابزار داخلی برای یافتن سریع‌ترین آی‌پی‌های گوگل جهت اتصال به رله.
- **تست اتصال**: بررسی سریع کیفیت و تاخیر (Latency) اتصال به سرور.
- **تجربه کاربری بومی**: طراحی شده با SwiftUI برای حداکثر بازدهی و زیبایی در macOS.

## نحوه کارکرد

Shade ترافیک شما را از طریق یک اسکریپت در زیرساخت گوگل (Apps Script) عبور می‌دهد. از آنجا که اتصال اولیه به دامنه `www.google.com` برقرار می‌شود، این روش در برابر فیلترینگ هوشمند (DPI) بسیار مقاوم است.

## شروع سریع

۱. **وارد کردن مشخصات**: مشخصات Script ID و Auth Key خود را وارد کنید.
۲. **شروع اتصال**: روی دکمه Start کلیک کنید.
۳. **تنظیم مرورگر**: مرورگر خود را روی پورت SOCKS5 یا HTTP تنظیم کنید (پیش‌فرض: 8081).
۴. **حالت سیستم**: با فعال کردن Set as system proxy، کل ترافیک سیستم را از تونل عبور دهید.

## جزییات فنی

- **پورت‌های محلی**: SOCKS5 (8081)، HTTP (8080) به صورت پیش‌فرض.
- **معماری**: پشتیبانی کامل از پردازنده‌های Apple Silicon و Intel.
- **امنیت**: تونل TLS 1.3 به زیرساخت‌های گوگل.
